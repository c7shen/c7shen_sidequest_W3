## Project Title

Side Quest Week 3: One Day Left Interactive Story

---

## Description

> This project is an interactive narrative game about making decisions within a single day.
> The user progresses through 3 time periods: morning, afternoon, evening. At each stage, the user makes a choice that affects hidden variables: stress, energy, clarity, and connection.
> The story ends with a reflective outcome based on user decisions.

---

## Interaction Instructions

- Click INSTRUCTION button or press 'i' to read instruction
- Click BACK button or press 'b' to return to menu
- Click START button or press 'enter' to begin
- Read the story text on each screen
- Make a choice by either clicking on button or pressing number keys 1, 2, 3
- Click anywhere or press 'r' on ending screen to restart
